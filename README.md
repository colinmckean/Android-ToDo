# Android-ToDo

Android ToDo is a task management list app developed for android written in Java using android studio as an IDE.
Compile and run in android studio and install on AVD or your own development device.

Initial todo screen will show tasks that are in the status of ToDo, you can change to view inpress and done by using the menu in the top right.
View all lists by clicking the View Lists from here you can add new lists and in turn add tasks to these lists.

Long tap on the new list button will remove a list.
